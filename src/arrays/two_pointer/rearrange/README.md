# 🔄 Partitioning Two Pointers

## Recognition Clues

Think **Partitioning** when:

- ✅ Rearrange elements in-place
- ✅ Move all X to one side and Y to the other
- ✅ Constant extra space required
- ✅ Order of elements is not important

---

## Pointer Rule

Move pointers **independently**.

```
Left correct  → move left
Right correct → move right
Both wrong    → swap
```

Do **NOT** move both pointers on every iteration.

- ✅ Pair with Target Sum
- ✅ Rearrange 0s and 1s
